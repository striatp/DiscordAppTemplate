## ğŸ› ï¸ Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/Striatp/DiscordAppTemplate.git
   cd DiscordAppTemplate
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Configure the bot:**

   - Add your **Discord bot token** inside a `.env` file:

   ```txt
   DISCORD_TOKEN=Your_Token_Here
   ```

4. **Start the bot:**

   ```sh
   npm run start
   ```
